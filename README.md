# SienaARChemistry
A Unity project for making chemistry simulations in augmented reality.


Instructions:

Windows: Open Git Bash
MacOS: Open Terminal

Run the following commands:
  - mkdir ~/UnityProjects
  - cd ~/UnityProjects
  - git clone https://github.com/JustinMarotta13/SienaARChemistry.git
  - cd SienaARChemistry
  - git status
